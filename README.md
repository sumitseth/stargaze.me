stargaze.me
---
Simple, easy way to star Github repos.

## Tech used

* Apollo GraphQL for accessing the Github GraphQL API
* React + Hooks
* Jest + React-testing-library
* Hosted on Netlify
* CI/CD Pipeline
* Prettier + Eslint
* Styled Components


## Running tests

```bash
$ npm run test
```
